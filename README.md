# Day2-ITI-Git

## How to remove dev branch locally and remotely
#### remove locally
git push remoteName :branchName
#### remove remotely
git branch :branchName

## How to list tags
git tag --list

## How to delete tag locally and remotely
#### remove locally
git tag -d tagName
#### remove remotely
git push remoteName --delete tagName
